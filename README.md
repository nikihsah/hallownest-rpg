# Hallownest RPG — Foundry VTT System

Независимая неофициальная система для Foundry VTT 13. Проект создаётся с нуля; в качестве исследования использован публичный прототип [`sergey-kznetsov/hk-rpg-foundry`](https://github.com/sergey-kznetsov/hk-rpg-foundry).

## Состояние

Версия `0.1.0` содержит:

- Actor типа `bug`;
- базовые характеристики и ресурсы;
- d6-пул с успехами на 5–6;
- предметы нескольких категорий;
- русскую локализацию и первоначальное оформление.

## Установка для разработки

Разместите репозиторий в `FoundryVTT/Data/systems/hallownest-rpg`, перезапустите Foundry и создайте мир на системе **Hallownest RPG — Unofficial System**.

Проект не связан с Team Cherry и Foundry Gaming LLC. Официальные графические и звуковые материалы Hollow Knight в репозиторий не входят.

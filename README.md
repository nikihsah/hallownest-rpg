# Hallownest RPG — Foundry VTT System

Независимая неофициальная система для Foundry VTT 13. Проект создаётся с нуля; в качестве исследования использован публичный прототип [`sergey-kznetsov/hk-rpg-foundry`](https://github.com/sergey-kznetsov/hk-rpg-foundry).

## Состояние

Версия `0.6.0` содержит:

- Actor типа `bug`;
- главные и вторичные характеристики по HK-Kit;
- встроенный справочник из 15 военных и мистических Путей HK-Kit;
- каталог из 133 Черт и Подчерт HK-Kit с категориями и автоматическими модификаторами;
- Голод, Сытость, Жуть, Привлекательность, Метки, Припасы и Эссенцию;
- Черты и Подчерты как полноценные Item-документы Foundry;
- пошаговое создание: сначала шаблон жука, затем черты и снаряжение;
- автоматическое применение модификаторов активных черт;
- отдельные вкладки обзора, черт, амулетов, техник, умений, инвентаря и заметок;
- автоматические тесты ключевой логики создания персонажа;
- инициативу как сумму d6 Грации;
- автоматический размер токена и учёт Скорости при движении в бою;
- d6-пул с успехами на 5–6;
- предметы нескольких категорий;
- русскую локализацию и первоначальное оформление.

## Установка

В Foundry VTT откройте **Setup → Game Systems → Install System**, вставьте ссылку в поле **Manifest URL** и нажмите **Install**:

```text
https://raw.githubusercontent.com/nikihsah/hallownest-rpg/main/system.json
```

После установки создайте новый мир и выберите систему **Hallownest RPG — Unofficial System**.

### Ручная установка для разработки

Разместите репозиторий в `FoundryVTT/Data/systems/hallownest-rpg`, перезапустите Foundry и создайте мир на системе **Hallownest RPG — Unofficial System**.

Проект не связан с Team Cherry и Foundry Gaming LLC. Официальные графические и звуковые материалы Hollow Knight в репозиторий не входят.
